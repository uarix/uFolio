## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/uarix/uFolio.git

# 进入项目目录
cd uFolio

# 安装依赖
npm i

# 启动开发服务器
npm run dev
```

## 🛠 开发环境配置

### 系统要求

- Node.js (v14 或更高版本)
- npm (v6 或更高版本)
- Git

### 配置说明

自定义你的网站：
- 在 `/src/configs/` 目录下修改常规设置
- 在对应目录下调整 i18n 文件进行本地化
